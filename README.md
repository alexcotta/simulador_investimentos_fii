# Simulador Investimentos FII

Projeto em Excel para simular investimentos em Fundos Imobiliários (FIIs), usando fórmulas básicas como SOMA, SUBTRAÇÃO, MÉDIA e referências de células. Permite inserir dados de rentabilidade, dividendos e risco para calcular retornos projetados e otimizar alocação. Aplica análise de dados do Excel para determinar quanto investir e quais FIIs priorizar, facilitando decisões financeiras assertivasv
